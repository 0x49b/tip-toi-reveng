Here you see an Example of a tttool-YAML for the syntax-highlighter prism.

Usually Yaml-Code looks like this:


comment: This is a Testyaml, only to test Colors, not to build a gme-File
init: $mode:=0 
scripts:

  ERSTENS:
  - $modus==0? P(kein_modus)
  - $modus==1? P(blob) P(modus_1)
  - $modus==2? P(blob) P(modus_2)
  - $modus!=0? $modus!=1? $modus!=2? P(ooops)

Instructions

Step 1: Get Prism

Download prism.js and prism.css here: http://prismjs.com/download.html. There you can choose all the languages you like to show on your Website.

Step 2: Your HTML

Integrate the four files in your HTML like this:

<!DOCTYPE html>
<html>
<head>
	...
	<link href="prism.css" rel="stylesheet" />
	<link href="prism-yamltttool.css" rel="stylesheet" /> 
</head>
<body>
	...
	<script src="prism.js"></script>
	<script src="prism-yamltttool.js">
</body>
</html>

Step 3: Place your Yaml-Code

Wrap your Yaml-Code with <pre> and <code class="language-yamltttool"> Tags.

<!DOCTYPE html>
<html>
<head>
	...
	<link href="prism.css" rel="stylesheet" />
	<link href="prism-yamltttool.css" rel="stylesheet" /> 
</head>
<body>

<pre><code class="language-yamltttool">
media-path: media/sounds_%s
language: de
welcome: willkommen
product-id: 42
comment: This is a Testyaml, only to test Colors, not to build a gme-File
init: $mode:=0

  ERSTENS:
  - $modus==0? P(kein_modus)
  - $modus==1? P(blob) P(modus_1)
  - $modus==2? P(blob) P(modus_2)
  - $modus!=0? $modus!=1? $modus!=2? P(ooops)
</code></pre> 
  
	<script src="prism.js"></script>
	<script src="prism-yamltttool.js">
</body>
</html>

Thats all.

Visit the prism-site for more information about Highlighting.
